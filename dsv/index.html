This file documents Data Specification Vocabulary (DSV).

Overview

Classes

Application Profile

IRI https://w3id.org/dsv#ApplicationProfile
Label Application Profile
Definition An Application Profile identifies classes and properties from vocabularies, and class profiles and property profiles from Application profiles that are re-used in a certain usage context. It also allows specification editors to further adjust the description of re-used classes and properties for that usage context without the need to explicitly create subclasses and subproperties.

Application Profile Specification

IRI https://w3id.org/dsv#ApplicationProfileSpecification
Label Application Profile Specification
Definition An Application Profile Specification contains resources defining and describing an Application Profile, including the human readable documentation, machine readable definition, etc.

Invalid Entity Profile

IRI https://w3id.org/dsv#InvalidEntityProfile
Label Invalid Entity Profile
Definition Invalid Entity Profile is a profile, which breaks profile compliance rules. This may be for a good reason such as temporary, to be propagated upstream, inconsistency.
Subclass of Entity Profile

Property Profile

IRI https://w3id.org/dsv#PropertyProfile
Label Property Profile
Definition Profile of a property allows specification editors to specify profile specific name, definition, usage note, domain, range and cardinality.
Subclass of Entity Profile
Subclasses Datatype Property Profile, Object Property Profile

Property Value Reuse

IRI https://w3id.org/dsv#PropertyValueReuse
Label Property Value Reuse
Definition Specifies, which values of which properties of which resources are reused for the current Profile. E.g. reuse of profile name, reuse of class name, reuse of profile definition, reuse of profile usage note, etc.

Vocabulary Specification

IRI https://w3id.org/dsv#VocabularySpecification
Label Vocabulary Specification
Definition A Vocabulary Specification consists of resources defining and describing a Vocabulary, e.g. definitions of classes and properties, typically with their future re-use in application profiles in mind.
Subclass of Resource Descriptor
